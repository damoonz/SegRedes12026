## Descripción
What does asm1(0x3ef) return? Submit the flag as a hexadecimal value (starting with '0x'). NOTE: Your submission for this question will NOT be in the normal flag format.
## Solución 
Se hace operación hex considerando el valor dado y se le resta 0x9

```
└─$ python                   
Python 3.13.9 (main, Oct 15 2025, 14:56:22) [GCC 15.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> hex(0x3ef - 0x9)
'0x3e6'

```

0x3e6
