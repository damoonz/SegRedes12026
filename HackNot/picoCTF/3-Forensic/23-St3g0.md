## Descripción
Download this image and find the flag.

- [Download image](https://artifacts.picoctf.net/c/216/pico.flag.png)
## Solución 
Se descarga la imagen indicada y se utiliza la herramienta zteg, utilizada para ver información oculta de pngs, al poner el siguiente comando en nuestra terminal se muestra la bandera:
```
┌──(moonz㉿damoonz)-[~/Documentos/picoctf/forensic]
└─$ zsteg -a pico.flag.png 
b1,rgb,lsb,xy       .. text: "picoCTF{7h3r3_15_n0_5p00n_a1062667}$t3g0"
b1,abgr,lsb,xy      .. text: "E2A5q4E%uSA"

...

```
